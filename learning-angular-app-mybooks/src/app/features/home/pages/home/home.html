Welcome to the Home Page

This is a sample Angular application demonstrating routing, feature modules, and reusable components.

For project purposes, the theme of this application is a personal book manager called myBooks.

View myBooks or Create Book Entry
  • Angular Router with feature modules
  • Route parameters for dynamic navigation
  • Route guards for authentication
  • Reusable UI components
  • Add new books with title, author, and series info
  • Track reading status for each entry
  • View book entries in a table and open details
  • Quickly edit book entries from the details page
